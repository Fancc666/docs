# 摄像头

![](https://s2.loli.net/2025/06/18/rEzDcaV4NetPguQ.png)

[https://www.565455.xyz/test/cam/](https://www.565455.xyz/test/cam/)

这是一个能测试摄像头好坏的工具。我通过这个项目学会利用浏览器调用设备的摄像头和麦克风。

- 点击`开启(env)`开启的是后置摄像头。
- 点击`开启(usr)`开启的是前置摄像头。
- 点击`关闭`按钮能停止视频捕获。

```{note}
网站并不会把你的视频音频文件上传到服务器，内容都在本地浏览器渲染，请放心使用！
```
